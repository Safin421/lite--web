How to change the package name?

You can change it easily from Android Studio. Here are the steps:

  • In the Project pane, click on the little gear icon. Uncheck/Deselect the Compact Empty Middle Packages option.
  • Your package directory will now be broken up in individual directories.
  • Individually select each directory you want to rename, and: Right-click itSelect RefactorClick on Rename in the Pop-up dialog.
  • Click on Rename Package instead of Rename Directory Enter the new name and hit Refactor and allow a minute to let Android Studio update all changes.
  • Now open your build.gradle (Usually 'app' or 'mobile'). Update the applicationId to your Package Name and Sync Gradle, if it hasn't already been updated automatically.
  • Open Build > Clean Project (wait for the build process to complete then do next step)
  • Open Build > Rebuild Project (wait for the build process to complete)
  • That's it, Done..

How to change app name?

  • Open Android Studio > Source Code> res > values > strings.xml
  • Enter your app name inside "app_name" string tag:
    <string name="app_name">Your App Name HERE</string>

How to change Splash Screen, Application Icon and Other icons?

How To Change Application Icon

  • Open Android Studio > app > res > all mipmap folder > ic_launcher.png and ic_launcher_round.png
  • Rename your square icon with ic_launcher.png and Round icon with ic_launcher_round.png paste in mipmap folder

How To Change Splash Screen ICON

  • Open Android Studio > app > res > drawable > splash_icon.png
  • Rename your splash screen image with splash_icon.png and paste in drawable folder


How to Change AdMob Details?

  • Android Studio > Source Code> res > values > strings.xml
  • Enter your admob app id inside "admob_app_id" string tag:
    <string name="admob_app_id">ca-app-pub-6953442522646328~5693407931</string>
  • Android Studio > Source Code> App> Values >String>
  • Enter your Admob Banner ad id inside "ADMOB_SMALL_BANNER_AD_ID" string tag:
    public static final String ADMOB_SMALL_BANNER_AD_ID ="ca-app-pub-6953442522646328/7253901615";
  • Enter your Admob interstitial ad id inside "ADMOB_INTER_AD_ID" string tag:
    public static final String ADMOB_INTER_AD_ID ="ca-app-pub-3940256099942544/1033173712";
  • If you want to Disable Ads :
    Replace isAdsEnabled = true to false like below line.
    public static boolean isAdsEnabled = false;

How to Generate Signed APK?

Create New Keystore

  • Click Build > Generate Signed APK

  • Select APK and click Next

  • Click On Create New or if you are updating app click choose existing

  • Select Keystore Path
  • Enter Keystore password and confirm password
  • Enter any name for alias
  • Enter key password and confirm password
  • Choose Validity Years
  • Fill rest of the details and click OK

  • Enter keystore password, alias name and key password and click next

  • Choose the signed apk destination
  • Tick the signatures
  • Click Finish

Note :- Please save the keystore and remember the keystore password, alias and key password. If you want to update app in future you have to use this keystore. With new keystore you cannot update your app
